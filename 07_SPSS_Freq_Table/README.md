# Exercise 7: Frequency Table (SPSS)

### Objective
To present raw data as a frequency table using SPSS to understand the distribution of variables.

### Steps in SPSS
1. **Input Data**: Enter your data in the *Data View* (e.g., a list of ages or marks).
2. **Analyze**: Go to `Analyze` -> `Descriptive Statistics` -> `Frequencies`.
3. **Select Variable**: Move the variable you want to analyze from the left list to the `Variable(s)` box.
4. **Statistics/Charts**: (Optional) Click `Charts` to select a bar chart or histogram.
5. **Run**: Click `OK`.

### Expected Output Format
The Output Viewer will show a table like this:

| Age | Frequency | Percent | Valid Percent | Cumulative Percent |
|-----|-----------|---------|---------------|--------------------|
| 18  | 5         | 25.0    | 25.0          | 25.0               |
| 19  | 8         | 40.0    | 40.0          | 65.0               |
| 20  | 7         | 35.0    | 35.0          | 100.0              |
| **Total** | **20** | **100.0** | **100.0** | |

### Explanation
- **Frequency**: Number of times the value occurs.
- **Percent**: Percentage of total cases.
- **Cumulative Percent**: Running total of the percentage.

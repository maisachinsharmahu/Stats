# Exercise 8: Finding Outliers (SPSS)

### Objective
To identify data points that differ significantly from other observations in a dataset.

### Steps in SPSS (Boxplot Method)
1. **Analyze**: Go to `Analyze` -> `Descriptive Statistics` -> `Explore`.
2. **Dependent List**: Move your continuous variable to the `Dependent List` box.
3. **Plots**: Click the `Plots` button, ensure `Boxplots` (Factor levels together) is selected.
4. **Run**: Click `OK`.

### How to Identify Outliers
- In the generated **Boxplot**, SPSS marks outliers with small circles (o) or asterisks (*).
- Points marked with **o** are "Out" values (1.5 to 3 times the IQR).
- Points marked with **\*** are "Extreme" values (more than 3 times the IQR).

### Expected Output
The Boxplot will show the median, quartiles, and individual points outside the whiskers representing the outliers.

| Statistic | Value |
|-----------|-------|
| Median    | 50.0  |
| Minimum   | 10.0  |
| Maximum   | 150.0 (Outlier) |

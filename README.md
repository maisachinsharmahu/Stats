# Statistics Practical Portfolio (Scilab, SPSS, R)

This repository contains 14 advanced statistical exercises implemented across Scilab, SPSS, and R. Each exercise includes the code, steps, and visual output.

---

### Scilab Exercises (Matrix & Calculus)
1.  **[Matrix Operations](./01_Matrix_Ops/README.md)**: Basic addition, subtraction, multiplication, and transpose.
2.  **[Eigenvalues](./02_Eigen_Values/README.md)**: Spectral decomposition of square matrices.
3.  **[Equation Solvers](./03_Equations_Solvers/README.md)**: Gauss Elimination, Jordan, and Seidel methods.
4.  **[Matrix Properties](./04_Matrix_Properties/README.md)**: Verification of associative, commutative, and distributive laws.
5.  **[Row Echelon Form](./05_Row_Echelon/README.md)**: Matrix reduction using `rref`.
6.  **[Functions & Derivatives](./06_Functions_Derivatives/README.md)**: Plotting f(x) and its derivatives.

### SPSS Exercises (Descriptive & Business Stats)
7.  **[Frequency Table](./07_SPSS_Freq_Table/README.md)**: Presenting raw data distributions.
8.  **[Outliers Identification](./08_SPSS_Outliers/README.md)**: Using Boxplots to find extreme values.
9.  **[Project Risk Analysis](./09_SPSS_Risk_Analysis/README.md)**: Comparing projects using Coefficient of Variation.

### R Programming Exercises (Data Science & Analytics)
10. **[Scatter & Residual Plots](./10_R_Scatter_Plots/README.md)**: Diagnostics and influential data points.
11. **[Correlation Analysis](./11_R_Correlation/README.md)**: Pearson correlation and significance testing.
12. **[Time Series](./12_R_Time_Series/README.md)**: Trend and seasonality analysis of monthly data.
13. **[Linear Regression](./13_R_Regression/README.md)**: Predictive modeling with `lm()`.
14. **[Probability Distributions](./14_R_Probability/README.md)**: Normal distribution visualization and calculation.
